# WebIT Solution Patient Management System Project

## 📌 Project Description

This project is a **Patient Management System** developed using **ASP.NET MVC, SQL Server, and Stored Procedures**.
It allows users to perform **CRUD operations (Create, Read, Update, Delete)** on patient records.

The system helps manage patient details such as admission date, investigation, room rent, and total bill.

---

## 🚀 Features

* Add new patient details
* View patient list
* Update patient information
* Delete patient records
* Calculate total amount automatically
* Use of SQL Server Stored Procedures

---

## 🛠 Technologies Used

* ASP.NET MVC (C#)
* SQL Server
* ADO.NET
* HTML
* CSS
* Bootstrap
* Stored Procedures

---

## 🗄 Database

The project uses **SQL Server database** with a stored procedure for handling CRUD operations.

Stored Procedure Name:
`spWebItPatient`

---

## ▶ How to Run the Project

1. Clone the repository
2. Open the project in **Visual Studio**
3. Restore database in **SQL Server**
4. Update the **connection string** in `Web.config`
5. Run the project
